Mutual Information and Channel Capacity

Mutual Information — Shared Uncertainty

The mutual information I(X; Y) measures how much knowing Y tells us about X (and vice versa):

I(X; Y) = H(X) − H(X | Y) = H(Y) − H(Y | X)

Equivalently, I(X; Y) = DKL(PXY ∥ PX ⊗ PY) — it is the KL divergence between the joint distribution and the product of marginals. I(X; Y) = 0 if and only if X and Y are independent. Mutual information is symmetric: I(X; Y) = I(Y; X). It is always non-negative.

The Information Venn Diagram

The entropy quantities form a Venn-diagram structure:

  • H(X) — total uncertainty about X
  • H(Y) — total uncertainty about Y
  • H(X, Y) — joint entropy; uncertainty about both together
  • H(X | Y) = H(X, Y) − H(Y) — remaining uncertainty about X after observing Y
  • I(X; Y) = H(X) + H(Y) − H(X, Y) — the overlap: information shared between X and Y

The chain rule H(X, Y) = H(X) + H(Y | X) generalizes: H(X1, …, Xn) = Σ H(Xi | X1, …, Xi-1).

Noisy Channels and Shannon's Channel Capacity Theorem

A discrete memoryless channel maps input X to output Y with transition probabilities P(Y | X). The channel capacity is:

C = maxP(X) I(X; Y)

Shannon's noisy channel coding theorem (1948): it is possible to transmit information reliably at any rate R < C bits per channel use, using sufficiently long block codes. Reliable transmission at rates R > C is impossible. For the binary symmetric channel with error probability p, C = 1 − H(p, 1−p) bits, where H is binary entropy.

Mutual Information vs. Correlation

Pearson correlation measures linear dependence only. Mutual information captures any statistical dependence, linear or nonlinear. Two variables can have zero correlation but high mutual information (e.g., Y = X² with X symmetric around zero: Corr(X, Y) = 0 but I(X; Y) > 0).

This makes mutual information valuable in feature selection, causal inference, and neuroscience — wherever the relationship between variables might be nonlinear. However, MI estimation from data is harder than estimating correlation, especially in continuous high-dimensional settings.

Applications — Feature Selection and Neural Codes

In machine learning, mutual information feature selection ranks features by I(feature; label), keeping features that share the most information with the target. Methods like mRMR (minimum Redundancy Maximum Relevance) also subtract mutual information between features to reduce redundancy.

In neuroscience, the neural coding efficiency question asks how much of the theoretical channel capacity H(stimulus) is actually transmitted by spike trains. Efficient coding theory (Barlow, 1961) predicts that sensory neurons should minimize redundancy and maximize information transmission — equivalent to maximizing I(response; stimulus) subject to metabolic constraints.

Data Processing Inequality and Sufficient Statistics

The data processing inequality (DPI): if X → Y → Z form a Markov chain (Z is a function of Y only, not X), then I(X; Z) ≤ I(X; Y). Processing data can never increase the information about the original source. Equality holds if and only if Z is a sufficient statistic for X given Y.

The DPI is fundamental to information theory and has concrete implications: no algorithm applied to sensor data can extract more information about the underlying state than the raw sensor captures; a neural network layer cannot increase the mutual information between its output and the target above that of its input (deep learning thus performs lossy compression, not information amplification); and channel cascades satisfy I(X; output) ≤ minlink C(link), where C is the channel capacity.

A sufficient statistic T(X) for a parameter θ satisfies I(θ; T(X)) = I(θ; X) — no information about θ is lost. The Neyman-Fisher factorization theorem characterizes sufficient statistics by whether the likelihood factors as g(T(x), θ) · h(x). The minimal sufficient statistic achieves maximum compression while preserving all information: it is the coarsest sufficient statistic, and the DPI guarantees that any further compression loses information about θ.
