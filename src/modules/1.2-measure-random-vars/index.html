About Measures & Random Variables

Measure Axioms

A probability measure P on a measurable space (Ω, F) is a function P: F → [0, 1] satisfying three axioms:

Non-negativity: P(A) ≥ 0 for all A ∈ F.
Normalization: P(Ω) = 1.
Countable additivity: For disjoint sets A1, A2, …, P(∪ Ai) = ∑ P(Ai).

On a finite sample space, specifying P({ω}) for each outcome ω ∈ Ω completely determines the measure, provided the values are non-negative and sum to 1.

Random Variables as Measurable Functions

A random variable X: Ω → &Reals; is a measurable function, meaning that for every Borel set B ⊆ &Reals;, the preimage X−1(B) = {ω ∈ Ω : X(ω) ∈ B} belongs to the sigma-algebra F.

On a finite space, every function X: Ω → &Reals; is automatically measurable with respect to the power set sigma-algebra. The key insight is the induced sigma-algebra σ(X), which captures exactly the information that X reveals about the outcome.

More atoms means X distinguishes more outcomes — the identity function generates the finest (power set) sigma-algebra, while a constant function generates the trivial one.

The Cumulative Distribution Function

The CDF of X is defined as FX(t) = P(X ≤ t) = PX((−∞, t]). It is a complete description of the distribution of X: two random variables with the same CDF have the same distribution, regardless of what probability space (Ω, F, P) they are defined on.

The CDF satisfies three key properties: (1) non-decreasing, (2) right-continuous with left limits (càdlàg), (3) FX(−∞) = 0 and FX(+∞) = 1. On a finite sample space, the CDF is a staircase function — each jump at value x has size PX({x}).

Conversely, any right-continuous non-decreasing function from &Reals; to [0,1] with the correct limits is the CDF of some measure — this is the Helly-Bray theorem and is how probability measures on (&Reals;, Β(&Reals;)) are constructed from CDFs without needing an explicit sample space.

Independence: A Measure-Theoretic View

Events A, B ∈ F are independent if P(A ∩ B) = P(A) · P(B). Random variables X, Y are independent if σ(X) and σ(Y) are independent sigma-algebras: for all A ∈ σ(X) and B ∈ σ(Y), P(A ∩ B) = P(A)P(B).

This is equivalent to: for all Borel sets B1, B2 ⊆ &Reals;, P(X ∈ B1, Y ∈ B2) = P(X ∈ B1) · P(Y ∈ B2). For discrete RVs, this means P(X = x, Y = y) = P(X = x) · P(Y = y) for all x, y.

The pushforward of (X, Y) jointly is a measure on (&Reals;², Β(&Reals;²)). Independence holds if and only if this joint pushforward equals the product measure PX ⊗ PY. The joint distribution factorizes as a product of marginals.

Absolute Continuity and Density Functions

A probability measure μ is absolutely continuous with respect to Lebesgue measure λ (written μ ≪ λ) if λ(A) = 0 implies μ(A) = 0. By the Radon-Nikodym theorem, this is equivalent to the existence of a non-negative measurable function f (the probability density function) such that:

μ(A) = ∫A f(x) dλ(x) for all A ∈ Β(&Reals;)

The density f is the Radon-Nikodym derivative dμ/dλ. For continuous distributions like Normal or Exponential, f is the familiar PDF. For discrete distributions, no density exists with respect to Lebesgue measure — instead, the pushforward measure is absolutely continuous with respect to counting measure.
