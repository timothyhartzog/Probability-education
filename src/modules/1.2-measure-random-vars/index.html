About Measures & Random Variables

Measure Axioms

A probability measure P on a measurable space (Ω, F) is a function P: F → [0, 1] satisfying three axioms:

Non-negativity: P(A) ≥ 0 for all A ∈ F.
Normalization: P(Ω) = 1.
Countable additivity: For disjoint sets A1, A2, …, P(∪ Ai) = ∑ P(Ai).

On a finite sample space, specifying P({ω}) for each outcome ω ∈ Ω completely determines the measure, provided the values are non-negative and sum to 1.

Random Variables as Measurable Functions

A random variable X: Ω → &Reals; is a measurable function, meaning that for every Borel set B ⊆ &Reals;, the preimage X−1(B) = {ω ∈ Ω : X(ω) ∈ B} belongs to the sigma-algebra F.

On a finite space, every function X: Ω → &Reals; is automatically measurable with respect to the power set sigma-algebra. The key insight is the induced sigma-algebra σ(X), which captures exactly the information that X reveals about the outcome.

More atoms means X distinguishes more outcomes — the identity function generates the finest (power set) sigma-algebra, while a constant function generates the trivial one.
