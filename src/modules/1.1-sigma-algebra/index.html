About Sigma-Algebras

Definition

A sigma-algebra (or σ-field) F on a set Ω is a collection of subsets of Ω satisfying: (1) Ω ∈ F, (2) if A ∈ F then Ac ∈ F, and (3) if A1, A2, ... ∈ F then ∪ Ai ∈ F.

These closure properties ensure that any "reasonable" set operation applied to measurable sets yields another measurable set. The pair (Ω, F) is called a measurable space.

Generated Sigma-Algebras

Given any collection S of subsets, σ(S) denotes the smallest sigma-algebra containing all sets in S. It is constructed by taking all complements, unions, and intersections until closure is achieved. This module shows that process step by step.

Atoms and Partitions

On a finite set, every sigma-algebra is generated by a unique partition of Ω into atoms — the minimal non-empty sets in F. The number of atoms determines the "information content" of the sigma-algebra: more atoms means finer resolution.

Information and Filtrations

A filtration F0 ⊆ F1 ⊆ ... is a nested sequence of sigma-algebras representing information accumulating over time. A finer sigma-algebra (higher in the lattice) corresponds to more information. This is fundamental to conditional expectation and martingale theory.

Countable vs. Finite Closure — Why It Matters

An algebra (field) is closed under only finite unions. A sigma-algebra is closed under countably infinite unions. The extra power costs nothing on finite Ω — every finite union is trivially countable — but is essential in analysis.

On the real line, events like "X < r for all rational r in (a, b)" require a countable intersection of events, one per rational. Without countable closure the collection {X < r} for each r could belong to F while the limit event {X ≤ b} = ∩r>b, r∈ℚ{X < r} does not. Sigma-algebras guarantee this works.

The distinction also matters for continuity of measure: if An ↑ A then P(An) → P(A). Proving this uses countable additivity of σ-algebras, which finite algebras cannot deliver.

The Borel Sigma-Algebra on &Reals;

The most important sigma-algebra in probability is the Borel σ-algebra Β(&Reals;), defined as the smallest σ-algebra containing all open intervals (a, b). Equivalently, Β(&Reals;) = σ({(−∞, x] : x ∈ &Reals;}).

Every open set, closed set, countable intersection of open sets (Gδ), or countable union of closed sets (Fσ) belongs to Β(&Reals;). Random variables are defined as Borel-measurable functions: X: (Ω, F) → (&Reals;, Β(&Reals;)), ensuring that {ω: X(ω) ≤ x} ∈ F for all x ∈ &Reals; — which is exactly what allows P(X ≤ x) to be well-defined.

Non-Borel sets exist (proven using the axiom of choice) but are pathological — they cannot be given a consistent probability. The Lebesgue σ-algebra is a larger completion of Β(&Reals;) that includes all measure-zero sets and their subsets, but for probability theory Β(&Reals;) suffices.

Closure Size: How Many Sets Can a σ-Algebra Have?

On a finite set Ω with |Ω| = n, the power set has 2n subsets. Any sigma-algebra F satisfies |F| = 2k for some k ≤ n, where k is the number of atoms (the minimal non-empty elements of F). This means sigma-algebras on finite sets have sizes 2, 4, 8, …, 2n — only powers of two are possible!

Given g generators, the generated sigma-algebra can be at most 22g in size before stabilizing (since g generators create at most 2g atoms).
