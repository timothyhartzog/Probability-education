About the Borel-Cantelli Lemmas

The Limsup of Events

Given a sequence of events A1, A2, … the limsup (or limit superior) is the set of outcomes ω that belong to infinitely many of the An:

We write {An i.o.} (read "An infinitely often") for this event. An outcome is in the limsup if and only if, no matter how far along the sequence we go, it eventually appears in some later event.

First Borel-Cantelli Lemma (BC1)

If the total probability mass spread over the events is finite, then the probability of being in infinitely many of them is zero. This holds for any sequence of events — no independence required. Intuitively, if the events become improbable fast enough, they cannot keep occurring.

Second Borel-Cantelli Lemma (BC2)

If the sum diverges and the events are mutually independent, then with probability 1, infinitely many of the events occur. The independence condition is essential — without it, the divergence of the sum alone does not guarantee infinitely many occurrences (use the independence toggle to see this).

Why Independence Matters

When events are dependent, BC2 can fail. For example, if every event An is actually the same event A with P(A) = 1/n (which doesn't even make sense for a fixed event), or more concretely, if An = A for all n where P(A) > 0, then Σ P(An) = ∞ but {An i.o.} = A, which need not have probability 1. This module lets you toggle independence off to see dependent constructions where the limsup probability drops below 1 despite a divergent sum.
