Modes of Convergence

The Four Modes — Strongest to Weakest

A sequence of random variables Xn can converge to a limit X in several genuinely distinct senses. From strongest to weakest:

  • Almost sure (a.s.) convergence — Xn(ω) → X(ω) for almost every sample path ω. The set of paths that fail to converge has probability zero.
  • Lp convergence — E[|Xn − X|p] → 0. The p-th moment of the error vanishes. Implies convergence in probability.
  • Convergence in probability — P(|Xn − X| > ε) → 0 for every ε > 0. The chance of being far from the limit vanishes, but individual paths can still oscillate.
  • Convergence in distribution — Fn(x) → F(x) at all continuity points of F. Only the law converges; Xn and X need not live on the same space.

The hierarchy is: a.s. ⟹ in probability ⟹ in distribution. Lp convergence also implies convergence in probability. None of the converses hold in general.

Typewriter Sequence — A Key Counterexample

The typewriter sequence shows that convergence in probability does not imply almost sure convergence. Encode each integer n as a pair (m, k) where n corresponds to the k-th interval of length 1/2m in [0,1]. Let Xn = 1 on that interval, 0 elsewhere.

Then P(Xn = 1) = 1/2m → 0, so Xn → 0 in probability. But for any fixed ω ∈ [0,1], Xn(ω) = 1 infinitely often as the intervals sweep back and forth — so Xn(ω) does not converge to 0 for any ω.

Almost Sure vs. In Probability

Convergence in probability says: at each time n, the set {|Xn − X| > ε} shrinks. But new bad sets can keep appearing at later times.

Almost sure convergence requires P(lim supn {|Xn − X| > ε}) = 0 — the bad sets eventually stop appearing. A sufficient condition via Borel-Cantelli: if Σ P(|Xn − X| > ε) < ∞, then Xn → X almost surely.

Uniform Integrability and Lp

Lp convergence requires not just that values concentrate near X, but that the tails of the distribution also go to zero. The technical condition is uniform integrability: supn E[|Xn|p 1{|Xn| > M}] → 0 as M → ∞.

Vitali's convergence theorem: if Xn → X in probability and {Xn} is uniformly integrable, then Xn → X in L1. This generalises the Dominated Convergence Theorem, which provides a sufficient condition for uniform integrability via a dominating function.

Convergence in Distribution and the Portmanteau Theorem

Convergence in distribution (weak convergence) Xn ⟶d X means Fn(x) → F(x) at all continuity points of F. It is the weakest mode — it only says the laws converge, not the random variables themselves. Remarkably, there are many equivalent formulations:

Portmanteau theorem: the following are all equivalent to Xn ⟶d X:

  • E[f(Xn)] → E[f(X)] for every bounded continuous f
  • limsup P(Xn ∈ F) ≤ P(X ∈ F) for every closed set F
  • liminf P(Xn ∈ G) ≥ P(X ∈ G) for every open set G
  • P(Xn ∈ B) → P(X ∈ B) for every Borel B with P(X ∈ ∂B) = 0

The CLT is a theorem about convergence in distribution: (Sn − nμ)/(σ√n) ⟶d N(0,1). The individual random variables Sn/√n need not be close to any particular normal variable — only their distributions are close.

Continuous Mapping Theorem and Slutsky's Theorem

The Continuous Mapping Theorem (CMT): if Xn converges to X in any mode (a.s., probability, or distribution), and g is a continuous function (or merely continuous at the support of X), then g(Xn) converges to g(X) in the same mode. This lets you transfer convergence through transformations: if Xn ⟶d N(0,1), then Xn2 ⟶d χ2(1).

Slutsky's theorem handles sums and products: if Xn ⟶d X and Yn ⟶p c (a constant), then:

  • Xn + Yn ⟶d X + c
  • Xn Yn ⟶d cX

Note: Slutsky requires one sequence to converge to a constant. If Yn ⟶d Y (non-degenerate) and Xn ⟶d X, the joint distribution (Xn, Yn) need not converge to (X, Y) without an independence assumption. This is why the delta method (g(Xn) ≈ g(μ) + g′(μ)(Xn−μ)) applies: it combines CMT with Slutsky to derive the asymptotic distribution of smooth functions of estimators.
