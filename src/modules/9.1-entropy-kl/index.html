Entropy and Information Theory

Shannon Entropy — Measuring Uncertainty

The Shannon entropy of a discrete distribution P = (p1, …, pk) is:

H(P) = −Σ pi log2 pi

Entropy measures the average number of bits needed to encode a symbol drawn from P under the optimal code. A uniform distribution over k outcomes has maximum entropy H = log2 k bits — maximum uncertainty. A deterministic distribution (pi = 1 for one outcome) has H = 0 bits — no uncertainty. The convention 0 log 0 = 0 is enforced by continuity.

KL Divergence — Measuring Distributional Distance

The Kullback-Leibler divergence from Q to P measures the extra bits needed when coding P-distributed symbols using a code optimized for Q:

DKL(P ∥ Q) = Σ pi log(pi / qi)

Properties: DKL ≥ 0 always (Gibbs' inequality), with equality iff P = Q. It is not symmetric — DKL(P ∥ Q) ≠ DKL(Q ∥ P) in general. It is not a metric. In Bayesian inference, DKL(posterior ∥ prior) measures information gained from observing data. In variational inference, minimizing DKL(q ∥ p) approximates a complex posterior p with a tractable q.

Cross-Entropy and the Coding Interpretation

The cross-entropy H(P, Q) = −Σ pi log qi is the expected code length when symbols drawn from P are encoded with a code optimal for Q. It decomposes as:

H(P, Q) = H(P) + DKL(P ∥ Q)

The KL divergence is thus the excess cost of using the wrong distribution Q. This decomposition explains why cross-entropy loss is used in machine learning classification: minimizing H(P, Q) over parameters of Q is equivalent to minimizing DKL(P ∥ Q) since H(P) is independent of Q.

Jensen's Inequality and Information Inequalities

Most fundamental inequalities in information theory follow from Jensen's inequality: for a convex function φ, E[φ(X)] ≥ φ(E[X]). Since −log is convex, applying Jensen to the ratio p/q gives DKL(P ∥ Q) = EP[log(p/q)] ≥ log(EP[p/q]) = log 1 = 0.

The data processing inequality states that applying any function f to a random variable X can only reduce or preserve information: H(f(X)) ≤ H(X). This is the formal reason why lossy compression cannot be undone and why transforming data cannot increase its information content.

Differential Entropy for Continuous Distributions

For continuous distributions, the discrete sum becomes an integral: h(X) = −∫ f(x) log f(x) dx. Unlike discrete entropy, differential entropy can be negative and is not invariant under reparameterization. Adding a constant c to X increases h by log|c| (for scaling by c).

The Gaussian distribution N(μ, σ²) maximizes differential entropy among all distributions with fixed variance: h = ½ log(2πeσ²). This is the maximum entropy principle — the Gaussian makes no additional assumptions beyond mean and variance, which is why it arises naturally in so many contexts.
