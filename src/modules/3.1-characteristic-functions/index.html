About Characteristic Functions

Definition

The characteristic function of a random variable X is φX(t) = E[eitX] for t ∈ R. Unlike the moment generating function, it always exists.

Key Properties

φ(0) = 1 always. If X is symmetric about 0, φ is real-valued. Moments appear as derivatives: E[Xk] = φ(k)(0)/ik. φ uniquely determines the distribution (inversion theorem).

Connection to Fourier Transform

The characteristic function is the Fourier transform of the distribution. The inversion formula recovers the density (when it exists): f(x) = (1/2π) ∫ e-itx φ(t) dt.

Even when no density exists (e.g., for the Cantor measure or discrete distributions), the characteristic function is still well-defined and continuous. This makes it far more versatile than the PDF. The Lévy continuity theorem exploits this: φn(t) → φ(t) pointwise if and only if Xn converges in distribution to X.

Moments from the Characteristic Function

If E[|X|k] < ∞, the k-th moment can be read off the characteristic function via differentiation at t = 0:

E[Xk] = i-k φ(k)(0)

Concretely: E[X] = -i φ'(0) and Var(X) = -φ''(0) - (φ'(0))2. This is visible in the Show moment markers view: the slope at t=0 encodes the mean, and the curvature encodes the variance. Distributions with infinite variance (Cauchy) have characteristic functions that are non-differentiable at 0 — note the cusp in the modulus plot.

The full Taylor series at t = 0, when it converges, gives the moment generating function evaluated on the imaginary axis: φ(t) = M(it) = ∑k=0∞ E[Xk] (it)k/k!.

Lévy's Continuity Theorem and the Proof of the CLT

The cleanest proof of the Central Limit Theorem uses characteristic functions. Let X1, X2, … be i.i.d. with mean 0 and variance 1. The characteristic function of the standardized sum Zn = (X1 + … + Xn) / √n is:

φZn(t) = [φX(t/√n)]n

Taylor expanding φX(t/√n) = 1 - t2/(2n) + o(1/n) gives [1 - t2/(2n)]n → e-t2/2 as n → ∞, which is the characteristic function of N(0,1). By Lévy's continuity theorem, pointwise convergence of characteristic functions implies convergence in distribution.

Stable Distributions and Infinite Divisibility

A distribution is stable if a sum of i.i.d. copies has the same distribution up to centering and scaling. Normal, Cauchy, and Lévy distributions are all stable. The characteristic function of a stable distribution has the Lévy-Khintchine form:

log φ(t) = iμt - c|t|α(1 - iβ sgn(t) tan(πα/2))

where α ∈ (0, 2] is the stability index. α = 2 gives the normal distribution; α = 1 gives Cauchy. When α < 2 the variance is infinite; when α ≤ 1 even the mean is undefined. These are the only possible limit distributions for sums of i.i.d. variables.
