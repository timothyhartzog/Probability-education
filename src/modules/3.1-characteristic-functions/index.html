About Characteristic Functions

Definition

The characteristic function of a random variable X is φX(t) = E[eitX] for t ∈ R. Unlike the moment generating function, it always exists.

Key Properties

φ(0) = 1 always. If X is symmetric about 0, φ is real-valued. Moments appear as derivatives: E[Xk] = φ(k)(0)/ik. φ uniquely determines the distribution (inversion theorem).

Connection to Fourier Transform

The characteristic function is the Fourier transform of the distribution. The inversion formula recovers the density (when it exists): f(x) = (1/2π) ∫ e-itx φ(t) dt.
