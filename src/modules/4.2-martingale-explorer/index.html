Doob's Martingale Convergence Theorem

A submartingale Mn (where E[Mn+1 | Fn] ≥ Mn) that is bounded in L1 — i.e., supn E[|Mn|] < ∞ — converges almost surely to a finite limit M∞.

The intuition: if the expected value cannot keep growing unboundedly, the process must "settle down" almost everywhere. The proof uses the upcrossing lemma: for any interval [a, b], the expected number of times the process crosses from below a to above b is bounded by (E[|Mn|] + |a|)/(b-a).

For the simple random walk (a martingale), the condition sup E[|Mn|] < ∞ fails — the walk is recurrent and does not converge. Adding an absorbing barrier converts it to a bounded martingale that does converge (to 0 or the barrier level a.s.).

Sub- and Supermartingales — Drift and Applications

A process with E[Mn+1 | Fn] ≥ Mn is a submartingale (favorable game — expected value increases). E[Mn+1 | Fn] ≤ Mn defines a supermartingale (unfavorable game). Every martingale is both.

A biased random walk with step probability p > 1/2 is a submartingale. But the process Mn = Sn - n(2p-1) removes the drift, restoring the martingale property. Finding such a "compensating" sequence is the core idea of the Doob-Meyer decomposition: every submartingale decomposes as a martingale plus a predictable non-decreasing process.

Applications: the price process of a stock under the risk-neutral measure is a martingale (no free lunch). Defective systems and insurance loss processes are supermartingales (expected value decreases over time). The Optional Stopping Theorem in each case has different economic interpretations.

Martingale Transforms and Financial Derivatives

If Mn is a martingale and Hn is a predictable process (Hn is measurable with respect to Fn-1), then the martingale transform (H • M)n = ∑k=1n HkΔMk is also a martingale.

This is the discrete-time version of the stochastic integral ∫ H dM. In finance, Hk represents the number of shares held at time k (chosen based on information up to k-1 — no look-ahead), and Mk is the discounted asset price. The no-arbitrage condition is exactly the statement that the gain process ∫ H dM is a martingale, preventing systematic profit without risk.

The optional stopping theorem then implies that no predictable trading strategy can generate a positive expected profit: E[(H • M)T] = (H • M)0 = 0 for any stopping time T satisfying the integrability conditions.
