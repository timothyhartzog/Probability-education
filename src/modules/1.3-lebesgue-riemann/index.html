About Lebesgue vs. Riemann Integration

The Key Difference

Riemann integration partitions the domain [a,b] into subintervals and approximates the function's values on each. Lebesgue integration partitions the range of f and measures the preimage of each slice.

When They Agree

For bounded functions on [a,b], the Riemann integral exists if and only if the set of discontinuities has Lebesgue measure zero (Lebesgue's criterion). When it exists, both integrals give the same value. The Lebesgue integral is strictly more general.

The Dirichlet Function

The indicator of the rationals, 1Q(x), is the canonical example where Riemann integration fails but Lebesgue integration succeeds. Every subinterval contains both rationals and irrationals, so the Riemann upper sum is 1 and the lower sum is 0, regardless of partition size. But the Lebesgue integral is 0 because Q has measure zero.
