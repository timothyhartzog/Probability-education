About Lebesgue vs. Riemann Integration

The Key Difference

Riemann integration partitions the domain [a,b] into subintervals and approximates the function's values on each. Lebesgue integration partitions the range of f and measures the preimage of each slice.

When They Agree

For bounded functions on [a,b], the Riemann integral exists if and only if the set of discontinuities has Lebesgue measure zero (Lebesgue's criterion). When it exists, both integrals give the same value. The Lebesgue integral is strictly more general.

The Dirichlet Function

The indicator of the rationals, 1ℚ(x), is the canonical example where Riemann integration fails but Lebesgue integration succeeds. Every subinterval contains both rationals and irrationals, so the Riemann upper sum is 1 and the lower sum is 0, regardless of partition size. But the Lebesgue integral is 0 because ℚ has measure zero.

From the Lebesgue perspective, the range of 1ℚ is {0, 1}. The preimage of {1} is ℚ ∩ [0,1], which has Lebesgue measure 0. The preimage of {0} is (&Reals; ∖ ℚ) ∩ [0,1], which has measure 1. So the integral is 1 · 0 + 0 · 1 = 0.

Simple Function Approximation

The Lebesgue integral is constructed in stages. A simple function is a finite linear combination of indicator functions: φ = ∑i ai · 1Ai. Its integral is defined as ∫ φ dμ = ∑i ai μ(Ai).

For any non-negative measurable f, there exists a sequence of simple functions φn ↑ f pointwise. The Lebesgue integral is then defined as the supremum of ∫ φ dμ over all simple φ ≤ f. This construction guarantees ∫ f dμ is well-defined (possibly +∞) for all non-negative measurable f.

The Lebesgue slicing shown on the right panel is exactly this: each horizontal strip [y, y+δy) defines a set A = f−1([y, y+δy)), and the contribution is y · μ(A). Summing these gives the simple function approximation to ∫ f dμ.

Limit Theorems: MCT and DCT

The Lebesgue framework's greatest advantage is that limits can be passed inside integrals under mild conditions. Suppose fn → f pointwise:

  • Monotone Convergence Theorem (MCT): If 0 ≤ fn ↑ f, then ∫ fn dμ → ∫ f dμ. No absolute bound required — monotonicity is enough.
  • Dominated Convergence Theorem (DCT): If |fn| ≤ g for some integrable g, then ∫ fn dμ → ∫ f dμ. The function g dominates the sequence.
  • Fatou's Lemma: ∫ lim inf fn dμ ≤ lim inf ∫ fn dμ (for non-negative fn). The inequality can be strict if probability mass escapes to infinity.

These theorems fail for the Riemann integral. The Riemann integral of a pointwise limit of integrable functions may not equal the limit of integrals, even for bounded sequences on compact intervals. The Lebesgue framework resolves this, enabling the foundations of Lp spaces, Fourier analysis, and probability theory.

Lp Spaces and Integrability Classes

The Lebesgue integral defines a family of function spaces. For p ≥ 1, the space Lp(μ) consists of all measurable functions with finite p-norm: ‖f‖p = (∫ |f|p dμ)1/p < ∞.

  • L1: Integrable functions. Expectation E[X] = ∫ X dP is defined for L1 random variables.
  • L2: Square-integrable functions. Variance Var(X) = E[X2] − (E[X])2 is finite. L2 is a Hilbert space with inner product ⟨f, g⟩ = ∫ fg dμ.
  • L∞: Essentially bounded functions. ‖f‖∞ = ess sup |f| < ∞.

The Hölder inequality ∫ |fg| dμ ≤ ‖f‖p ‖g‖q (where 1/p + 1/q = 1) and the Minkowski inequality ‖f+g‖p ≤ ‖f‖p + ‖g‖p make Lp a Banach space. For probability: if X ∈ Lp then X ∈ Lq for all q ≤ p.
