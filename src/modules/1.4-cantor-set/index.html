About the Cantor Set

Construction

The Cantor set is constructed by starting with [0,1] and iteratively removing the open middle third of each remaining interval. At step n, we have 2n intervals each of length (1/3)n. The Cantor set C is the intersection of all these stages: C = ∩ Cn.

Remarkable Properties

The Cantor set is: uncountable (it contains all numbers in [0,1] whose ternary expansion uses only digits 0 and 2), closed and perfect (every point is a limit point), yet has Lebesgue measure zero (total length removed = 1).

Its Hausdorff dimension is log(2)/log(3) ≈ 0.631, strictly between 0 and 1.

The Cantor Function

The Cantor function (devil's staircase) is continuous, non-decreasing, maps [0,1] onto [0,1], yet has derivative equal to zero almost everywhere (on the complement of the Cantor set, which has full measure). It is the CDF of the Cantor measure, a singular continuous measure.
