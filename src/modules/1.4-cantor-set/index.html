About the Cantor Set

Construction

The Cantor set is constructed by starting with [0,1] and iteratively removing the open middle third of each remaining interval. At step n, we have 2n intervals each of length (1/3)n. The Cantor set C is the intersection of all these stages: C = ∩ Cn.

Remarkable Properties

The Cantor set is: uncountable (it contains all numbers in [0,1] whose ternary expansion uses only digits 0 and 2), closed and perfect (every point is a limit point), yet has Lebesgue measure zero (total length removed = 1).

Its Hausdorff dimension is log(2)/log(3) ≈ 0.631, strictly between 0 and 1.

The Cantor Function

The Cantor function (devil's staircase) is continuous, non-decreasing, maps [0,1] onto [0,1], yet has derivative equal to zero almost everywhere (on the complement of the Cantor set, which has full measure). It is the CDF of the Cantor measure, a singular continuous measure.

Singular continuous measures sit in between: they are neither discrete (no atom has positive measure) nor absolutely continuous (no density function f with μ = f · λ exists). The Lebesgue decomposition theorem guarantees that any measure uniquely decomposes as μ = μac + μsc + μpp — absolutely continuous, singular continuous, and pure point (discrete) parts.

Cardinality vs. Measure — The Paradox

The Cantor set resolves a naive intuition: "uncountably many points should have positive measure." In fact, the Cantor set C is:

  • Uncountable: |C| = |[0,1]| = 2ℵ0 (the cardinality of the continuum). This follows from the ternary representation — each point in C corresponds to an infinite sequence of 0s and 2s, and such sequences biject with {0,1}&Naturals;.
  • Lebesgue measure zero: The total length of removed intervals is (1/3) + 2(1/9) + 4(1/27) + … = ∑n=0∞ (2/3)n · (1/3) = 1. So 100% of [0,1]'s length is removed, yet C remains.

This resolves as follows: measure and cardinality are fundamentally different notions. The rationals ℚ ∩ [0,1] are countable (measure zero is intuitive). The irrationals are uncountable (measure 1 is intuitive). But the Cantor set breaks this pattern: uncountable yet measure zero. Cardinality cannot predict measure.

Ternary Representation and the Digit Characterization

Every x ∈ [0,1] has a base-3 (ternary) expansion: x = ∑k=1∞ dk / 3k where each digit dk ∈ {0, 1, 2}. The middle-third removal at step 1 eliminates all x with d1 = 1. At step 2, all x with d2 = 1 (within each remaining interval) are removed. After all steps:

C = {x ∈ [0,1] : all ternary digits of x are in {0, 2}}

From this, the bijection with {0,1}&Naturals; is explicit: map each digit 0 → 0, 2 → 1. This gives an uncountable encoding, proving |C| = 2ℵ0. The Cantor function maps x ∈ C to the binary number formed by replacing each digit 2 with 1: this makes the function constant on each removed interval and continuous on C.

Fat Cantor Sets and Positive Measure

The removal fraction slider above lets you vary how much is removed at each step. If at step n you remove a fraction α of each remaining interval (with α < 1/2 to maintain two pieces), the total measure of Cα is:

λ(Cα) = limn→∞ (1 − α)n = 0 when 0 < α ≤ 1/2

But a different construction — the Smith-Volterra-Cantor (fat Cantor) set — removes smaller and smaller fractions: remove 1/4 at step 1, 1/16 at step 2, 1/64 at step 3, etc. (total removed: ∑ 2n−1/4n = 1/2). The resulting set has measure 1/2, is nowhere dense, has empty interior, yet is uncountable and perfect. Such sets demonstrate that "nowhere dense" and "measure zero" are independent properties.
