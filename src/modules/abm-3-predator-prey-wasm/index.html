WebAssembly Predator-Prey Agent-Based Model

Same predator-prey ecosystem as the D3 module, but the simulation engine runs in WebAssembly — compiled from AssemblyScript via npm run build:wasm. WASM is loaded lazily: only downloaded when this page is opened. Falls back silently to the JavaScript engine if the .wasm file is absent. The D3 rendering and analytics layers are identical to abm-2.
